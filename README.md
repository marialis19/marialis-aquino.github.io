# Marialis Aquino | Portfolio

Portfolio personal donde muestro mi perfil profesional, skills y proyectos como Desarrolladora Full Stack Junior.

🔗 **Ver sitio en vivo:** [marialis-aquino.github.io](https://marialis-aquino.github.io)

## Sobre este proyecto

Sitio desarrollado con HTML, CSS y Bootstrap, con diseño responsive y una paleta de colores personalizada. Incluye secciones de presentación, skills técnicas y proyectos destacados con enlaces a sus repositorios.

## Tecnologías utilizadas

- HTML5
- CSS3 (variables custom, diseño responsive)
- Bootstrap 5
- Bootstrap Icons
- Devicon

## Secciones

- **Sobre mí** — presentación y formación
- **Skills** — frontend, backend, herramientas, diseño, metodologías ágiles e ingeniería de software
- **Proyectos** — trabajos destacados con stack tecnológico y enlaces a GitHub / demo
- **Contacto** — email, LinkedIn y GitHub

## Cómo verlo localmente

```bash
git clone https://github.com/marialis19/marialis-aquino.github.io.git
cd marialis-aquino.github.io
```

Abrí `index.html` en tu navegador, o usá la extensión **Live Server** de VS Code para verlo con recarga automática.
